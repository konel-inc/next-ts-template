# Konel:Next.js - typescript Template

コネルNext.js + TypescriptのTemplate Projectです。
  - Templateのアップデート以外のコミット厳禁。
  - Templateのアップデートが必要な場合は、Issueに登録 > PRで改善点の提出 > コードレビューごマージ
